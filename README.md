# Таксометр — Telegram Mini App

Приложение для учёта заработка таксистов.

## Функции
- Ввод данных о смене (пробег, доходы, расходы)
- Автоматический расчёт чистой прибыли
- Статистика за день / неделю / месяц
- Тёмная и светлая тема
- Данные сохраняются через Telegram CloudStorage

## Запуск локально
```bash
npm install
npm run dev
```

## Технологии
- React 18 + TypeScript
- Vite
- Tailwind CSS
- Telegram Web App SDK
